Om de services te runnen moet je eerst de node packages installen, er zijn 2 verschillende Directorys
2. Frontend
3. Backend

Run in cmd het commando "npm i"
Per folder die hier boven staat

Maak een .env file aan in de frontend Directory. 
De inhoud staat in example.env. (Deze staat ook in de frontend Directory)

Als je dat hebt gedaan dan moet je in de backend gaan
hierin run je "node server.js"

Daarna ga je naar de frontend
hierin run je "npm run serve"
