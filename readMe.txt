Om de services te runnen moet je eerst de node packages installen, er zijn 3 verschillende 
1. In de main Directory
2. Frontend
3. Backend

Run in cmd het commando "npm i"
Per folder die hier boven staat

Maak een .env file aan in de main Directory. 
De inhoud staat in example.env.

Als je dat hebt gedaan dan moet je in de backend gaan
hierin run je "node server.js"

Daarna ga je naar de frontend
hierin run je "npm run serve"
